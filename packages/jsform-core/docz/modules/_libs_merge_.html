Const getUiSchemaKeyRecursion

  • getUiSchemaKeyRecursion(uiSchemaKeys: string[], parentSchemaPath: string): string
    • Defined in libs/merge.ts:20

    根据给出的parentKeys和uiSchemaKeys来获取uiSchema的key

    1. 遍历uiSchemaKeys，分别于parentKeys做合并
    2. 合并后的keys去仓库里面找，如果为找到则报错
    3. 判断找到的schema中是否带有 $ref
    4. 如果有$ref，则更改parentKeys为$ref的路径
    5. 如果没有，则更改parentKeys为当前合并的keys

    Parameters

    • uiSchemaKeys: string[]

      当前的keys

    • parentSchemaPath: string

      父亲的schemaPath

    Returns string

    返回的key

Const initMergeSchema

  • initMergeSchema(parent: UiSchema | undefined, schemaPath: string, uiSchemas: Array<UiSchema | string>, curSchema: FxJsonSchema): UiSchema[]
    • Defined in libs/merge.ts:189

    合并uiSchema

    1. 先判断uiSchema中是否存在*
    2. 如果没有*号，则遍历uiSchema，合并数据
    3. 如果存在号；先合并之前和*之后的uiSchema
    4. 遍历当前的schema： 如果是object，则遍历properties，然后合并数据 如果是array，则直接返回items，然后合并数据

    Parameters

    • parent: UiSchema | undefined

      父亲的uiSchema

    • schemaPath: string

      当前的schema路径

    • uiSchemas: Array<UiSchema | string>

      全部的uiSchemas

    • curSchema: FxJsonSchema

      当前的Schema

    Returns UiSchema[]

    合并后的uiSchemas
